# Sif-Player-State
Small WoW addon to display AFK and PVP states

Adds custom tags to Unhalted Unit Frames:
- AFK status [curafk] (displays "AFK" when unit is away from keyboard)
- PVP status [curpvp] (displays "PVP" when unit is flagged for PVP)
- PVP - AFK status [curstate] (displays either of the above or "PVP - AFK" if both are true)
